# Module 2: Fundamentals of Prompt Engineering

Welcome to Module 2! This module covers the core fundamentals of prompt engineering, teaching you how to structure effective prompts for optimal results.

## Module Overview

In this module, you will learn:
- The anatomy of an effective prompt
- The 5 key components of great prompts
- How to apply prompt patterns for testing scenarios
- Common mistakes to avoid
- How to iterate and refine prompts

## Lessons

1. [Basic Prompt Structure](./01-basic-prompt-structure.md)
   - The 5 components: Role, Context, Task, Format, Constraints
   - Prompt formulas and templates
   - Real-world examples for QA and development
   - Progressive prompting technique
   - Exercises for prompt construction

2. [Parameters and Settings](./02-parameters-and-settings.md)
   - Understanding temperature and other parameters
   - How settings affect outputs
   - Optimizing for different use cases

3. [Best Practices](./03-best-practices.md)
   - Prompt engineering best practices
   - Tips for effective prompting
   - Common pitfalls and how to avoid them

## Learning Objectives

By the end of this module, you will be able to:
- ✅ Construct effective prompts using the 5-component framework
- ✅ Apply prompt patterns for testing and development tasks
- ✅ Avoid common prompt engineering mistakes
- ✅ Iterate and refine prompts for better results
- ✅ Use parameters and settings effectively
